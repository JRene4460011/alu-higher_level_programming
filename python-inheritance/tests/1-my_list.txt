===========================
How to use 1-my_list.py
===========================

Import:
    >>> MyList = __import__('1-my_list').MyList

Check instantiation:
    >>> my_list = MyList()
    >>> type(my_list)
    <class '1-my_list.MyList'>

Check inherits from list:
    >>> issubclass(MyList, list)
    True

Check __str__:
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> print(my_list)
    [3, 1, 2]

Check append():
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> print(my_list)
    [5]

Check print_sorted():
    >>> my_list = MyList()
    >>> my_list.append(4)
    >>> my_list.append(1)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 3, 4]

Check print_sorted() does not modify original:
    >>> my_list = MyList()
    >>> my_list.append(4)
    >>> my_list.append(1)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 3, 4]
    >>> print(my_list)
    [4, 1, 3]

Check print_sorted() with negative numbers:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(-5)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [-5, -1, 3]

Check empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []
